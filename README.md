
Tudományos együttműködési hálózatok elemzése szakdolgozathoz készített python programpáros.
2 feladatatot végez el:
- Webscraper: MTMT (Magyar Tudományos Művek Tára) API (MyCite2) elérésével letölti az előre meghatározotott Excel táblázatban (melyben szerepel az oktató MTID-je (MTMT-n elérhető azonosítója) és a tanszéke) szereplők publikációs listáit.
- Halozat_keszitő: A lementett publikációs listákat betölti, feldolgozza és azokból olyan hálózatok létrehozására alkalmas állományokat hoz létre, melyben a hálózat pontjai közötti kapcsolatok alkotják annak gerincét (kapcsolati háló alapú hálózat-felépítés), az oktatók együttműködései (legalább két oktató szerepel egy publikáció publikálói névsorában) alapján.

A projekt a szakdolgozathoz készült, Excel adatbázist NEM tartalmazza.
A repo célja a projekt megörzése, potenciálisán referenciaként való bemutatáshoz. 
Futtatása Pycharm IDE-ben biztosított, Python 3.15.5/7 alatt íródott.
